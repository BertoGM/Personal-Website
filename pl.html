About the Project

As a Premier League fan and CS student, I wanted to combine two interests — football and data visualization. This dashboard lets you explore live and historical PL data in one place, without having to jump between different websites.

The goal was to build something that felt close to a real product like FotMob or Sofascore, while learning how to architect a full stack app with a real API, a proxy server, and interactive data visualizations.

What It Does
  • ▸ Live standings table with UCL, Europa, and relegation zone indicators
  • ▸ Fixtures and results with live score support
  • ▸ Top scorers leaderboard per season
  • ▸ Player stats scatterplot — appearances vs goals, assists, shots, or passes
  • ▸ Pitch heatmap showing positional zones per player
  • ▸ Season switcher — browse data from 2022 through 2024
How It's Built

The frontend is plain HTML, CSS, and JavaScript with Vega-Lite for the scatterplot visualization. The backend is a lightweight Node.js + Express server that proxies all requests to the API-Football API — keeping the API key hidden server-side and out of the browser.

The app is deployed on Render's free tier with environment variables managing secrets. A .env file keeps credentials out of the GitHub repo entirely.

Current Limitations
  • ⚠ The free API plan caps at 100 requests/day and only supports seasons 2022–2024 — no live 2025/26 data.
  • ⚠ Pitch heatmaps are simulated based on player position — real positional tracking data isn't available on the free tier.
  • ⚠ Some players are missing from the heatmap search — simultaneous squad requests get rate-limited, so only partial rosters load.
  • ⚠ Render's free tier spins down after inactivity — the first load after a while may take ~30 seconds to wake up.
What's Next

Planned improvements include upgrading to a paid API plan for current season data and real heatmap tracking, adding individual match detail pages, a team overview page with squad stats, and upgrading hosting to keep the server always-on. Eventually I'd like to add a player comparison feature and efficiency metrics like goals per 90 minutes.
