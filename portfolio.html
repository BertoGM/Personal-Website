About the Project

This is the site you're on right now. I built it from scratch without any frameworks or templates — just raw HTML, CSS, and JavaScript — as a way to own my presence on the web and practice front-end fundamentals outside of coursework.

The design goal was to strike a balance between visual personality and clean readability — something that felt polished without being over-engineered. Every spacing decision, type choice, and animation was done by hand.

What's Included
  • ▸ Multi-page site — Home, About, Projects, Contact
  • ▸ Dark theme with custom CSS variables and a subtle grid background
  • ▸ Smooth fade-in animations on page load
  • ▸ Individual project detail pages for each project
  • ▸ Responsive layout that works across screen sizes
  • ▸ Sticky frosted-glass nav bar with active state tracking
How It's Built

No frameworks, no build tools, no npm — just HTML files linked to a shared style.css. The whole site is deployed on GitHub Pages and loads in under a second with zero JavaScript dependencies.

Typography is set using Google Fonts — DM Serif Display for headings, Outfit for body text, and DM Mono for labels and code. Colors are managed through CSS custom properties for easy theming.

Design Decisions
  • → Dark background (#0d0d0f) to reduce eye strain and let the accent colors pop.
  • → Green (#7fff6e) and cyan (#5de8ff) accents to avoid the generic purple-gradient AI aesthetic.
  • → Serif + mono font pairing for a "technical editorial" feel — familiar but distinctive.
  • → Project cards intentionally minimal — content-first, no decorative clutter.
